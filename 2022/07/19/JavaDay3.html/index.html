2.5 程序流程控制

1. 顺序结构

程序从上到下逐行地执行，中间没有任何判断和跳转。

2. 分支结构

根据条件，选择性地执行某段代码。

  • 有if…else和switch两种分支语句。

3. 循环结构

根据循环条件，重复性的执行某段代码。

  • 有while、do…while、for三种循环语句。

注：JDK1.5之后提供了foreach循环，方便的遍历集合、数组元素。

  • 上一行运算符总优先于下一行。

1. 顺序结构

  • Java中定义成员变量时采用合法的前向引用。如：
    public class Test{
    int num1 = 12;
    int num2 = num1 + 2;
}

错误形式：
public class Test{
int num2 = num1 + 2；
int num1 = 12;
}

2. 分支结构

分支语句1： if-else语句

if语句三种格式：

1.

if(true){
  执行代码块；
 }

例句：

int i = 1;
int k = 3;

if(i == 1 && k != 2){
  //当if后小括号里为true，打样输出变量i
  System.out.println(i);
}

2.

if(条件表达式){
  执行代码块；
 }
 else{
  执行代码块；
 }

例句：

//判断一个数组是奇数还是偶数，输出结果
int i = 3;
if(i%2 == 0){
  System.out.println("偶数");
}else{
  System.out.println("奇数");
}

3.

if(条件表达式){
 执行代码块；
     }
     else if (条件表达式){
 执行代码块；
     }
      ……
      else{
 执行代码块；
      }

例句：

//如果是数字1，输出星期一，2输出星期二，3输出星期三，如果不是1,2,3任何数字，输出不知道
int i = 3；
if(i == 1){
  System.out.println("星期一");
}else if(i == 2){
  System.out.println("星期二");
}else if(i == 3){
  System.out.println("星期三");
}else{
  System.out.println("不知道")；
}
if-else语句应用举例
public class TestAge{
    public static void main(String args[]){
        int age = 75;
    if (age< 0) {
      //当年龄小于0时，输出不可能
          System.out.println("不可能！");
    } else if (age>250) {
      //当年龄大于250时，输出是个妖怪
          System.out.println("是个妖怪！");
    } else {
      //其他情况，输出具体年龄
          System.out.println(“人家芳龄 " + age +" ,马马乎乎啦！");
    }
  }
}
if-else嵌套使用：

if-else可以嵌套使用，就是if-else里还可以嵌套多个if-else语句。

if(){
  //这里是可以写任何代码段，包括if-else这种
}else{
   //这里是可以写任何代码段，包括if-else这种
}

分支结构2：switch语句

switch(变量){
case 常量1:
     语句1;
     break;

case 常量2:
     语句2;
     break;
… …
case 常量N:
     语句N;
     break;

default:
     语句;
     break;
}

例句：

//如果是数字1，输出星期一，2输出星期二，3输出星期三，如果不是1,2,3任何数字输出不知道
int i = 2;

switch(i){
  case 1://if(i == 1)
  System.out.println("星期一");
  break;
  case 2://else if(i == 2)
  System.out.println("星期二");
  break;
  case 3://else if(i == 3)
  System.out.println("星期三")；
  break;
  default://else
  System.out.println("不知道")；
  break;
}

switch语句应用

public class Test{
   public static void main(String args[]){
       int i = 1;
     switch (i) {
     case 0:
         System.out.println("zero");
         break;
     case 1:
         System.out.println("one");
         break;
     default:
         System.out.println("default");
         break;
         }
       }
     }

switch语句应用

public class Test{
    public static void main(String args[]){
        String season = “summer”;
      switch (season) {
      case “spring”:
        System.out.println(“春暖花开");
        break;
      case “summer”:
        System.out.println(“夏日炎炎");
        break;
      case “autumn”:
        System.out.println(“秋高气爽");
        break;
      case “winter”:
        System.out.println(“冬雪皑皑");
        break;
      default:
        System.out.println(“季节输入有误");
      break;
      }
    }
  }

switch语句有关规则

  • switch(表达式)中表达式的返回值必须是下述几种类型之一：byte，short，char，int，枚举，String；

  • case子句中的值必须是常量，且所有case子句中的值应是不同的；

  • default子句是可任选的，当没有匹配的case时，执行default

  • break语句用来在执行完一个case分支后使程序跳出switch语句块；如果没有break，程序会顺序执行到switch结尾

break是必须写的，不然程序无法跳出循环，会直接从case匹配位置往下执行到一个新的break或是结尾。

例题：

//利用switch把小写类型的char型转换输出大写，只转a，b，c，d，e ，其他输出other
char i ='a';
switch(i){
  case 'a':
    System.out.println("A");
    break;
  case 'b':
    System.out.println("B");
    break;
  case 'c':
    System.out.println("C");
    break;
  case 'd':
    System.out.println("D");
    break;
  case 'e':
    System.out.println("E");
    break;
  default:
    System.out.println("other");
    break;
}

switch和if语句的对比

if和switch语句很像，具体什么场景下，应用哪个语句呢？

如果判断的具体数值不多，而且符合byte、 short 、int、 char这四种类型。虽然两个语句都可以使用，建议使用swtich语句。因为效率稍高。

其他情况：对区间判断，对结果为boolean类型判断，使用if，if的使用范围更广。

当你不知道应该使用if还是switch时，都使用if就可以。
